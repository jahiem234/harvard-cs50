## Selected Projects

### DNA

Analyzes DNA sequences by comparing Short Tandem Repeats (STRs) to identify individuals.

---

### Recover

Recovers deleted JPEG images from a forensic memory card image using file signature analysis.

---

### Filter

Implements common image processing filters including:

- Grayscale
- Reflection
- Blur
- Edge Detection

---

### Speller

Builds a high-performance spell checker using a hash table and linked lists.

---

### Runoff

Simulates a ranked-choice voting election using the runoff voting algorithm.

---

### Plurality

Implements a plurality voting system that determines election winners from user input.

---

### Cash

Calculates the minimum number of coins required to make change using a greedy algorithm.

---

### Scrabble

Computes player scores based on Scrabble letter values.
